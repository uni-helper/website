| 选项 | 参数 | 简写 | 说明 |
|------|------|------|------|
| **模板** | `--template` | `-t` | 选择项目模板 |
| **TypeScript** | `--typescript` | `--ts` | 启用 TypeScript |
| **ESLint** | `--eslint` | `-e` | 集成 ESLint 代码规范 |
| **插件** | `--plugin` | `-p` | 添加开发插件 |
| **模块** | `--module` | `-m` | 集成功能模块 |
| **UI库** | `--ui` | `-u` | 选择 UI 组件库 |
| **GUI界面** | `--gui` | - | 启动图形化界面 |
| **信息** | `--info` | - | 显示项目信息 |
